On-time delivery optimized

A fully dynamic Excel workbook (built with VBA macros) that connects tasks, durations, dependencies, resources, and costs into one coherent view — updating schedules, flagging delays, and tracking budget in real time. The Critical Path Method (CPM) pinpoints which tasks drive the project end date so managers can intervene where it actually matters.

02

Methodology

Step 01
Work Breakdown Structure (WBS)
Decomposed the project into tasks with defined durations, predecessors, and owners — capturing finish-to-start and start-to-start dependencies.
Step 02
Critical Path Calculation
Forward and backward passes computed earliest/latest start-finish for each task. Tasks with zero float form the critical path — any delay there delays the whole project.
Step 03
Dynamic Gantt Chart
VBA macros and conditional formatting render an auto-updating Gantt that shows planned vs. actual progress and highlights critical-path tasks in a distinct colour.
Step 04
Resource & Budget Tracking
Each task is linked to resources and unit costs. The model aggregates labour spend, flags over-allocation, and compares cumulative cost against baseline as an early-warning system.
